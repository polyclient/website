## What is Poqk?

Poqk is a fast, extensible, cross-platform database management solution for SQL and NoSQL databases. It provides tools for administration, development, and integration with a focus on security, performance, and extensibility.


## Why Poqk?

Most alternatives are either complex to use, restrictive, or lock essential features behind "pro" subscriptions. Poqk is fully open-source and modular, allowing unrestricted customization, external integrations, and support for SQL and NoSQL databases without reliance on proprietary solutions.

## Getting Started

To begin using Poqk, follow these steps:

1. **Install Poqk** – Refer to the [installation guide](installation/system-requirements.md) for setup instructions.
2. **Connect to a database** – Refer to the [first connection guide](quick-start/first-connection.md) for instructions on how to connect to a database.
3. **Explore Plugins** – Extend functionality with official and community-developed plugins.

## Contributing

Poqk is an open-source project, and contributions are welcome. To get started:

1. Clone the repository from [https://github.com/poqk/poqk](https://github.com/poqk/poqk).
2. Install the development tools:
   - [Go >= 1.24](https://go.dev)
   - [Task](https://taskfile.dev)
   - [Bun](https://bun.sh) for GUI and documentation.
3. Run `task install` to install the project dependencies.
4. Run `task dev` to run the CLI.
5. Follow the [contributor's guide](https://github.com/poqk/poqk/blob/master/CONTRIBUTING.md) for more details.
